# Movie Review

#### A website to Rate different Movies!, 08/25/2017 

#### Carson Canales

## Description
_Allows users to browse and rate movies based on title and genre. Users may leave a detailed review of any film they have seen!_
* Java
* Spark
* Postman
* Handlebars
